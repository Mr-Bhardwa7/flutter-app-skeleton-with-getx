# Flutter App Skeleton With GetX

##### version: 0.0.1

A project structure is design on flutter with the getX state management tool

**Project Structure**

```
project
└─── lib
    └─── src
        └─── controller
            |   base_controller.dart
            |   ...
        └─── core
            └───theme
            └───utilities
        └─── routes
            |   app_pages.dart
            |   app_routes.dart
        └─── services
            |   app_exceptions.dart
            |   base_client.dart
            |   ...
        └─── views
            └───homeScreen
            └───splashScreen
            └───...
    │   main.dart
│   README.md
│   ...
```
