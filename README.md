# Flutter App Skeleton With GetX

##### version: 0.0.2

A project structure is design on flutter with the getX state management tool

```
project
└─── lib

            |   base_controller.dart
            |   permissison_controller.dart
            |   ...

            |    app_binding.dart

            |    fonts.style.dart

        └─── models
        └─── routes

            |   app_routes.dart
        └─── services
            |   app_exceptions.dart
            |   base_client.dart
            |   permission_service.dart
            |   ...
        └─── views
            └─── homeScreen
            └─── splashScreen
            └─── ...
    │   main.dart
│   README.md
│   ...
```
